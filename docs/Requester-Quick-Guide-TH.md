# คู่มือย่อผู้แจ้งซ่อมทั่วไป

`PowerCare.CM` ใช้สำหรับ **แจ้งซ่อม** และ **ติดตามสถานะงาน** โดย **ไม่ต้อง Login**

## 1. แจ้งซ่อม 3 ขั้นตอน

1. กด `แจ้งซ่อมทันที`
2. กรอกข้อมูลให้ครบ:
   - ชื่อผู้แจ้ง
   - หน่วยงาน
   - Category
   - Zone
   - ชื่อเครื่องจักร
   - หัวข้อปัญหา
   - รายละเอียดปัญหา
   - ความเร่งด่วน
3. กด `ส่งแจ้งซ่อม`

## 2. สิ่งสำคัญหลังแจ้งซ่อม

- ระบบจะแสดง **เลขที่แจ้งซ่อม**
- ให้ **จดเลขนี้ไว้ทันที** หรือถ่ายรูปหน้าจอเก็บไว้
- ตัวอย่างเลข:

`CM-2026-06-0001`

## 3. ติดตามสถานะงาน 2 ขั้นตอน

1. กด `ติดตามสถานะงาน`
2. กรอกเลขที่แจ้งซ่อม แล้วกด `ค้นหา`

## 4. ความหมายของสถานะที่พบบ่อย

- `รับเข้าระบบ` = ระบบรับรายการแล้ว
- `ระหว่างดำเนินการ` = มีผู้รับงานแล้วและกำลังทำงาน
- `รอตรวจรับ/ปิดงาน` = ช่างทำเสร็จและรอวิศวกรตรวจรับ
- `ปิดงานสำเร็จ` = งานเสร็จสมบูรณ์แล้ว
- `ยกเลิก` = งานนี้ถูกยกเลิก

## 5. ข้อควรจำ

- กรอกชื่อเครื่องจักรให้ชัดเจน
- อธิบายอาการเสียตามที่พบจริง
- เลือก Zone และ Category ให้ถูกต้องที่สุด
- อย่าปิดหน้าจอก่อนจดเลขที่แจ้งซ่อม

## 6. ถ้าไม่พบเลขที่แจ้งซ่อม

- ตรวจสอบว่ากรอกเลขถูกต้องหรือไม่
- ตรวจสอบเครื่องหมาย `-` ให้ครบ
- ถ้ายังไม่พบ ให้ติดต่อผู้ดูแลระบบหรือผู้เกี่ยวข้องในหน่วยงาน

---

**ลำดับการใช้งานสั้น ๆ:**
